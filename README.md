# Hepsiburada Test Cases

Bu repository, Hepsiburada case study kapsamında hazırlanan manuel test case çalışmalarını içermektedir.

## Kapsam

Çalışma, ürün detay sayfasındaki **“Yorum Yap”** akışına odaklanmaktadır.  
Test senaryoları, kullanıcının yorum yapma sürecindeki temel fonksiyonel akışları ve kritik validasyonları kapsayacak şekilde hazırlanmıştır.

## İçerik

Bu repository içerisinde aşağıdaki çalışmalar yer almaktadır:

- Manuel functional test case dokümanı
- Yorum yapma akışına ait end-to-end senaryolar
- Giriş yapmış / yapmamış kullanıcı senaryoları
- Ürünü satın almış / almamış kullanıcı senaryoları
- Yorum akışındaki yönlendirme ve validasyon kontrolleri

## Test Kapsamı

Hazırlanan test case’ler aşağıdaki alanları kapsamaktadır:

- Ürün detay sayfasından yorum akışının başlatılması
- Giriş yapmamış kullanıcının login sayfasına yönlendirilmesi
- Başarılı login sonrası yorum akışına devam edilmesi
- Ürünü satın almayan kullanıcı için uyarı mesajı doğrulaması
- Ürünü satın alan kullanıcı için yorum yapabilme akışı
- Değerlendirme alanları, yönlendirmeler ve temel iş kuralları

## Doküman Yapısı

Test case dokümanı aşağıdaki alanlardan oluşmaktadır:

- Test Case ID
- Test Scenario
- Preconditions
- Test Steps
- Expected Result
- Actual Result
- Status
- Priority
- Type
- Notes

## Amaç

Bu çalışmanın amacı, ilgili kullanıcı akışının fonksiyonel açıdan test edilebilirliğini göstermek, temel iş kurallarını doğrulamak ve manuel test tasarımı yaklaşımını ortaya koymaktır.
